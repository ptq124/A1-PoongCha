## 그라운드 룰
### 1. 스크럼 진행하기

1. 스크럼은 평일 10시에 진행한다.
2. 오늘 할 일을 공유한다.
3. 오늘 나의 상태를 공유한다.

### 2. 회고 진행하기

1. 회고는 평일 오후 6시에 진행한다.
2. 오늘 내가 한 일을 공유한다.
3. 개선할 점을 공유한다.
4. 내일 할 일을 공유한다.
 
### 3. 매주 월요일에 좋은 컨디션과 밝은 얼굴로 오기

## 스프린트 진행
- 스프린트는 1주일 단위로 나누어 정의한다.
- 마일스톤은 스프린트와 대응되게 1주일 작업 단위로 운영한다.
- 주차별 마일스톤을 미리 정의하고, 주차가 시작하기 전 세부 태스크들을 정의한다.
- 금토일 3일간 다음 주에 할 백로그 등록하기
